Project Name: ValleyFC

Soccer club player tracker

This application is a full-stack Django application inspired by my soccer club Valley FC. It persist of data in PostgreSQL, it autheticates users using Django's built-in authentication and it also implements authorization by restricting access to creation, updating, and deletion of sources. It serves a tracking tool and stat record keeper for a sports club.
